FAQ (Frequently Asked Questions) to Json converter - by Eric Gregori.

Usage: FAQtoJ.py -h | --help
       FAQtoJ.py -v | --version
       FAQtoJ.py -i FAQfile
       FAQtoJ.py -i FAQfile -o JSONfile


Options:
  -h,--help             : This help text
  -v,--version          : Version
  -i <FAQfile>          : input FAQ file as text
  -o <JSONfile>         : output JSON file
